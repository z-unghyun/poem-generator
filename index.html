시 생성기

글로벌문화1: AI시대의 시와 스토리텔링

시가 일상언어를 공격해 경험 기반의 언어적 도약을 만든다면, LLM은 확률분포에서 다음 단어를 고른다. 이 생성기는 동일한 Hugging Face 오픈소스 모델을 두 방식으로 작동시켜 평균적인 언어를 벗어나는 시를 실험한다.

생성 모드

슬라이더 값을 자연어 지시문으로 변환해 같은 모델에 전달합니다. 안정적이고 빠른 생성 방식입니다.

경험 밀도 경험어를 얼마나 강하게 반영할지 조절
언어 도약도 고확률 단어를 피하고 낯선 후보를 선택
다다 강도 반복, 절단, 비문, 우연적 조합의 정도
구체적인 장소, 감각, 감정이 들어갈수록 결과가 좋아집니다.
프롬프트: 경험 반영 지시 / 실험: 경험어 logit boost
프롬프트: 낯선 은유 지시 / 실험: 상위 확률 후보 회피
반복, 절단, 비문, 무작위 재배열을 허용하는 정도
생성 버튼을 누르면 이곳에 시가 나타납니다. Hugging Face backend가 연결되면 실제 모델 생성이 실행됩니다.
